# Conquest
Survive and conquest new lands using tools and weapons, gathering resources, building base, crafting items and fighting local creatures.
  
![](Screens/anim01.gif)

## Implemented features
### Inventory and Items
- picking
- dropping
- transfering
- consuming
- equipping
- customizable items using ScriptableObjects

### Combat
- custom weapon items and attacks using ScriptableObjects

### Other
- movement
- chopping trees
